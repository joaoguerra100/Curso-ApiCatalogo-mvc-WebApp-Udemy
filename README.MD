# 🌐 Web App MVC - Consumo da API de Categorias

Este projeto é uma aplicação ASP.NET Core MVC que consome a API de Categorias para exibir e gerenciar categorias de produtos.

## 🎯 Funcionalidades

- Listagem de categorias
- Criação, edição e exclusão de categorias
- Consumo de API externa via `HttpClientFactory`
- Interface responsiva com Bootstrap

## 🔗 Requisitos

- .NET 8 SDK
- API de Categorias rodando localmente ou hospedada
